About Project

The project "COLLEGE E-RESOURCE SYSTEM" deals with the idea of uploading data like previous year question papers, syllabuses etc. by the faculty members which will be available for the students to help them in the learning process. The students will be able to login with the help of login credentials provided to them by the admin and then they can download the e-resources. The main aim of the project is to make a hassle-free learning process for both faculty as well as the students. The faculty member can upload all the important resources which will be beneficial for the student and those resources can be accessed by the student any time throughout the semester.
